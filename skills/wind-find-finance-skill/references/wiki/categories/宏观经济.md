# 宏观经济

## 范围

中国宏观经济指标：GDP（总量 / 增速 / 结构）、CPI / PPI、社融 / M2、利率（LPR / DR007 / 国债收益率曲线）、PMI（制造业 / 非制造业 / 服务业）、进出口、工业增加值、固定资产投资、社会消费品零售总额。
