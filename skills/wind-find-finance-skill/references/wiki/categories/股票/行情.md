# 股票 · 行情

## 范围

A 股 / 港股实时报价、K 线（日 / 周 / 月，含前复权 / 后复权 / 不复权）、分钟级走势、万得板块成分。覆盖最新价、前收、开盘、最高、最低、涨跌、成交量、成交额。

| K 线（日 / 周 / 月，复权）| [wind-quote-skill](../../skills/wind-quote-skill.md) | `quote_get_kline` |

- "贵州茅台最新价是多少"
