# 第三方金融能力收录 · 占位

## 当前状态

**v1 wiki 仅收录 Wind AIMarket 平台已开放的自家 skill**，**暂不收录第三方**（包括但不限于：竞品厂商 / 数据服务商 / 行情商 / 量化平台等）。

## 为什么留这个占位文件

随着 Wind AIMarket 平台演进，未来可能会**有限度地收录**满足以下条件的第三方金融能力 skill：

- 与平台已开放能力**互补不冲突**（如外汇 / 美股 / 加密货币等当前 placeholder 资产类目）
- 通过 MCP 协议标准化封装，能直接被 wind-find-finance-skill 推荐
- 标识清晰的 vendor / markets / 能力性质，便于 AI 给用户陈述差异

未来收录第三方时，本文件会扩展为**收录原则 + 候选 skill 清单 + 决策让人做的差异表**。

## v1 阶段的"第三方"约束（写给未来的我们）

- ❌ **不点名**任何具体厂商（不写"Tushare / iFind / FactSet / Bloomberg ..."）
- ❌ **不做对标**（不写"国内独家"、"对标某产品"等市场表述）
- ✅ **诚实告知**：用户提到的第三方平台，AI 直接说"该能力可通过用户自行寻找第三方 skill 解决"，不替用户决策
- ✅ **保留扩展位**：候选 skill ≥2 时启用"差异表 + 决策让人做"模板（当前自家 skill 职责互补不冲突，模板预留未来生效）
